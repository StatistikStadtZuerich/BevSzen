# PopScen

## Population Scenarios

- Population scenarios for the city of Zurich.

- Code: This [R](https://www.r-project.org/) code is used to generate various population development scenarios for the city of Zurich.

- Data: Most imput data is [Open Data](https://data.stadt-zuerich.ch/).
